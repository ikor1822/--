# 📚 Стандартная библиотека

Gopi поставляется с набором встроенных (`VNative`) функций для работы со списками и вводом-выводом.

## Ввод-вывод (I/O)
* `print(x)` — выводит значение в стандартный поток вывода консоли. Поддерживает форматированный вывод массивов.
* `readFile(path)` — читает всё содержимое файла по указанному пути и возвращает его в виде строки.

## Работа со списками
Списки в Gopi создаются синтаксисом `[1, 2, 3]`. Склеивание списков происходит через бинарный оператор `+`.

* `len(list)` — возвращает количество элементов в списке.
* `head(list)` — возвращает первый элемент списка.
* `tail(list)` — возвращает список без первого элемента.

## Функции высшего порядка
* `map(func, list)` — применяет функцию `func` к каждому элементу списка и возвращает новый список.
* `filter(func, list)` — возвращает новый список, оставляя только те элементы, для которых `func(x)` вернула `true`.
* `fold(func, acc, list)` — сворачивает список, применяя функцию `func(acc, x)` слева направо.
