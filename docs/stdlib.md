# Стандартная библиотека (StdLib)

В интерпретатор встроено базовое окружение `env0`, содержащее нативные функции (`VNative`), написанные на F#. 

## Ввод-вывод
* `print(x)` — выводит значение в стандартный поток вывода. Умеет рекурсивно форматировать массивы, числа и строки.
* `input()` — считывает строку из стандартного потока ввода и возвращает ее как строковое значение.
* `readFile(path)` — читает содержимое файла по указанному пути и возвращает его как строку.
* `writeFile(path, text)` — записывает строку `text` в файл по указанному пути. Если файл не существует — создает его, если существует — перезаписывает.

## Структуры данных: Списки
Списки являются базовой структурой данных в Gopi. Они объявляются в квадратных скобках: `arr = [1, 2, 3]`. Поддерживается конкатенация через оператор `+`.

* `len(list)` — возвращает количество элементов списка.
* `head(list)` — возвращает первый элемент (голову).
* `tail(list)` — возвращает новый список, содержащий все элементы кроме первого (хвост).

## Функции высшего порядка
* `map(func, list)` — применяет функцию `func` к каждому элементу списка и возвращает новый массив.
* `filter(func, list)` — фильтрует список, оставляя только элементы, для которых функция `func` вернула `true`.
* `fold(func, acc, list)` — выполняет левостороннюю свертку списка, последовательно применяя функцию `func(acc, x)` к аккумулятору и текущему элементу.
