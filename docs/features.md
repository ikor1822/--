# ✨ Возможности языка Gopi

Язык Gopi тесно следует парадигме строгого функционального программирования. 

## 1. Переменные и Немутабельность
Связывание значений происходит с помощью ключевого слова `let`. Все значения в Gopi иммутабельны.
```javascript
let x = 10
let y = x * 2
```

## 2. Ленивые вычисления (Lazy Evaluation)
Главная фишка Gopi. Выражения вычисляются **только тогда, когда их значение действительно необходимо** (Call-by-need).
```javascript
// Деление на ноль вызовет ошибку, НО...
let bomb = 1 / 0 
let safe = 42

// Программа успешно выполнится, так как 'bomb' никогда не вызывается.
print(safe) 
```

## 3. Функции и Замыкания (Closures)
Gopi поддерживает функции как объекты первого класса. Они захватывают свое лексическое окружение при создании.
```javascript
let makeAdder = fun(x) {
    fun(y) { x + y }
}
let add10 = makeAdder(10)
print(add10(5)) // 15
```

## 4. Частичное применение (Currying)
Если вы передадите в функцию меньше аргументов, чем она ожидает, она не упадет с ошибкой, а вернет новое замыкание!
```javascript
func multiply(x, y, z) { x * y * z }

let multBy2 = multiply(2)
let multBy2And3 = multBy2(3)

print(multBy2And3(4)) // 24
```

## 5. Рекурсия
Для рекурсии используется ключевое слово `func`, которое связывает имя функции с ее телом внутри ее же локального окружения, позволяя функции вызывать саму себя бесконечно.
