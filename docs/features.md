# Основные особенности языка Gopi

Вычислительная модель Gopi построена вокруг нескольких ключевых функциональных концепций.

## 1. Неявное объявление переменных
В языке отсутствуют ключевые слова вроде `let`, `var` или `const`. Переменные объявляются путем прямого присваивания значения идентификатору. При этом все структуры данных остаются строго иммутабельными (неизменяемыми).
```javascript
x = 10
y = x * 2
```

## 2. Ленивые вычисления (Lazy Evaluation)
Важнейшая особенность языка. В Gopi используется стратегия вычислений Call-by-need. При присваивании переменной выражения оно не вычисляется сразу. Вместо этого создается отложенное вычисление.

Оно будет выполнено только в тот момент, когда значение переменной станет строго необходимым (например, при передаче в функцию `print` или при использовании в условном операторе `if`).

```javascript
// Переменная zero содержит ошибку (деление на ноль).
// Однако программа не завершится с ошибкой, так как вычисление отложено.
zero = 1 / 0 
safe = 42

print(safe) // Успешно выведет 42. Переменная zero проигнорирована.
```

## 3. Функции первого класса и Замыкания
Функции в Gopi являются полноправными объектами: их можно передавать аргументами, возвращать из функций и сохранять в массивы. При создании функция захватывает текущее лексическое окружение (образует замыкание).

* Анонимные функции объявляются через `fun`.
* Именованные функции объявляются через `func`.

```javascript
makeAdder = fun(x) {
    fun(y) { x + y }
}
add10 = makeAdder(10)
print(add10(5)) // Вывод: 15
```

## 4. Частичное применение (Currying)
Если при вызове функции передать меньше аргументов, чем указано в её сигнатуре, интерпретатор автоматически вернет новое замыкание, ожидающее недостающие параметры.
```javascript
func multiply(x, y, z) { x * y * z }

multBy2 = multiply(2)
print(multBy2(3, 4)) // Вывод: 24
```

## 5. Рекурсия
Поскольку в языке нет циклов (`for`, `while`), все повторяющиеся операции выполняются с помощью рекурсии. Для этого реализовано ключевое слово `func`. 

В отличие от анонимных лямбд `fun`, конструкция `func` неявно связывает имя функции с её собственным телом внутри её же лексического окружения. Это позволяет функции безопасно вызывать саму себя.

```javascript
func fact(n) {
    if n == 0 { 
        1 
    } else { 
        n * fact(n - 1) 
    }
}

print(fact(5)) // Вывод: 120
```
