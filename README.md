# Язык программирования Gopi

**Gopi** — это интерпретируемый функциональный язык программирования, разработанный на F# в рамках учебного проекта. Язык является полным по Тьюрингу и обладает строгой статической областью видимости.

Синтаксис языка намеренно сделан минималистичным (отсутствуют ключевые слова для объявления переменных, как в Python), однако вычислительная модель базируется на строгих функциональных концепциях: ленивых вычислениях, замыканиях и функциях первого класса. 

Интерпретатор написан с нуля без применения автоматических генераторов парсеров.

## 🚀 Режимы работы интерпретатора

Интерпретатор поддерживает два режима работы. Для запуска необходим установленный **.NET SDK**.

### 1. Интерактивный режим (REPL)
Если запустить интерпретатор без указания файла, откроется интерактивная среда (Read-Eval-Print Loop).
```bash
dotnet fsi gopi.fsx
```

### 2. Файловый режим
Для выполнения готового исходного кода передайте путь к файлу в качестве аргумента:
```bash
dotnet fsi gopi.fsx example.gopi
```

## 👥 Команда разработки
Проект выполнен командой из 3 человек:
1. **[Имя 1]** — Разработка абстрактного синтаксического дерева и парсера
2. **[Имя 2]** — Разработка интерпретатора, рефализация ленивых вычислений.
3. **[Имя 3]** — Стандартная библиотека, написание примеров, реализация repl, и документация.
