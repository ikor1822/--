# Язык программирования Gopi

**Gopi** — это интерпретируемый функциональный язык программирования, разработанный на F#.
Синтаксис языка был сделан похож на Python + Go.


## 🚀 Режимы работы интерпретатора

Интерпретатор поддерживает два режима работы. Для запуска необходим установленный **.NET SDK**.

### 1. Интерактивный режим (REPL)
Если запустить интерпретатор без указания файла, откроется интерактивная среда (Read-Eval-Print Loop).
```bash
dotnet fsi gopi.fsx
```

### 2. Файловый режим
Для выполнения готового исходного кода передайте путь к файлу в качестве аргумента:
```bash
dotnet fsi gopi.fsx example.gopi
```
